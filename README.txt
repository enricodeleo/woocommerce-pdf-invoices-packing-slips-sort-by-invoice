Sort orders by invoice number within the admin screen on WooCommerce

== Description ==

This plugin extends WooCommerce PDF Invoices & Packing Slips (https://github.com/wpovernight/woocommerce-pdf-invoices-packing-slips) making the invoice number columns sortable.

A small but very handy improvement to this great plugin by __@wpovernight__ (all credits for the main plugin to them).
